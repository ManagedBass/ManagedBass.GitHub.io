ConnectedFiltersProcedure Delegate

User defined callback function to receive connected filters (see ChannelGetConnectedFilters(Int32, ConnectedFiltersProcedure, IntPtr)).

Namespace: ManagedBass.DShow
Syntax
public delegate bool ConnectedFiltersProcedure(IntPtr Filter, string FilterName, bool HasPropertyPage, IntPtr User);
Parameters
Filter
IntPtr

A pointer to the connected filter instance (IBaseFilter). Note that this instance should be used inside the callback as it is released automaticaly once the callback returned.
FilterName
String

The name of the connected filter.
HasPropertyPage
Boolean

true if the filter has a property page (dialog), false if not.
User
IntPtr

The user instance data given when ChannelGetConnectedFilters(Int32, ConnectedFiltersProcedure, IntPtr) was called.
Returns
Boolean

true to continue, else false.
