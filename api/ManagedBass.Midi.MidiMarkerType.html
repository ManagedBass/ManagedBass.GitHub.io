MidiMarkerType Enum

The Marker type, to be used with StreamGetMark(Int32, MidiMarkerType, Int32, out MidiMarker).

Namespace: ManagedBass.Midi

Copyright

Copyright notice (MIDI meta event 2).

Value: 6

CuePoint

Cue events (MIDI meta event 7).

Value: 1

InstrumentName

Instrument name events (MIDI meta event 4).

Value: 8

KeySignature

Key signature events (MIDI meta event 89). That gives the key signature (in Text) in the form of "a b", where "a" is the number of sharps (if positive) or flats (if negative) and "b" signifies major (if 0) or minor (if 1).

Value: 5

Lyric

Lyric events (MIDI meta event 5).

Value: 2

Marker

Marker events (MIDI meta event 6).

Value: 0

Text

Text events (MIDI meta event 1).

Value: 3

Tick

FLAG: get position in ticks instead of bytes

Value: 65536

TimeSignature

Time signature event (MIDI meta event 88). The time signature events are given in the form of "numerator/denominator metronome-pulse 32nd-notes-per-MIDI-quarter-note", eg. "4/4 24 8".

Value: 4

TrackName

Track name events (MIDI meta event 3).

Value: 7
