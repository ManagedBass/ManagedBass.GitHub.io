Reference

The comment reference. Together with CommentType this indicates to what the comment refers. If CommentType=General then this should be 0. If CommentType=Channel then 0 = all channels, 1 = 1st channel, 2 = 2nd channel, etc. If CommentType=SoundSource then 0 = DSD recording, 1 = analogue recording, 2 = PCM recording. If CommentType=FileHistory then 0 = general remark, 1 = name of the operator, 2 = name or type of the creating machine, 3 = time zone information, 4 = revision of the file.
