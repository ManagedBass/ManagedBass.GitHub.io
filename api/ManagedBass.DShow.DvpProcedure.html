DvpProcedure Delegate

User defined DSP callback function (to be used with ChannelSetDVP(Int32, DvpProcedure, IntPtr)).

Namespace: ManagedBass.DShow
Syntax
public delegate void DvpProcedure(int Handle, int Channel, IntPtr Buffer, int Length, BassDShowDvpType DataType, int Width, int Height, IntPtr User);
Parameters
Handle
Int32

The DVP handle.
Channel
Int32

The video channel that the DVP applies to.
Buffer
IntPtr

The pointer to the sample data to apply the DVP to.
Length
Int32

The number of bytes to process.
DataType
BassDShowDvpType

One of the BassDShowDvpType values.
Width
Int32

The video width.
Height
Int32

The video height.
User
IntPtr

The user instance data given when ChannelSetDVP(Int32, DvpProcedure, IntPtr) was called.
