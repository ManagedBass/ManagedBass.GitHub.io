Remarks

On Windows, ID consists of a manufacturer identifier in the LOWORD and a product identifier in the HIWORD. This will not uniquely identify a particular device, ie. multiple devices may have the same value.. On OSX, ID is the device's "kMIDIPropertyUniqueID" property value, which is unique to the device. On Linux, id contains the device's ALSA client ID in the LOWORD and port ID in the HIWORD.
