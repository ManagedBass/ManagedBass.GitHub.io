WMATagFormat Enum

The "type" of the tag and values strings for use with EncodeSetTag(Int32, IntPtr, IntPtr, WMATagFormat).

Namespace: ManagedBass.Wma

Ansi

ANSI strings.

Value: 0

Binary

Write a binary tag.

The length of the binary data is given in the high word.

Value: 256

Unicode

Unicode (UTF-16) strings (recommended to be used with .Net).

Value: 1

Utf8

UTF-8 strings.

Value: 2
