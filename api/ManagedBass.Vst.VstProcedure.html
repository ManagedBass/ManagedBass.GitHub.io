Parameters

The action parameter, one of the BassVstAction values (see below).
Int32 Param1

The first parameter (see the VST SDK for further details).
Int32 Param2

IntPtr User

The user parameter as specified in the SetCallback(Int32, VstProcedure, IntPtr) call.
