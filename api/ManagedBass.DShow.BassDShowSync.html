BassDShowSync Enum

Channel Sync type to be used with ChannelSetSync(Int32, VideoSyncProcedure, IntPtr).

Namespace: ManagedBass.DShow

DvdChapterStart

Triggered when a new DVD chapter starts.

data: the new chapter index.

Value: 2

DvdError

Triggered when an error occured on a DVD stream.

data: alwqays 0.

Value: 4

DvdTitleChange

Triggered when a title change occured in a DVD stream.

data: new title number.

Value: 3

End

Triggered when the stream ends.

data: alwqays 0.

Value: 1

Repaint

Triggered by video renderers when it needs a repaint. A call to Un4seen.Bass.AddOn.DShow.BassDShow.BASS_DSHOW_ChannelRepaint(System.Int32,System.IntPtr,System.IntPtr) should be done, but not necessary.

data: alwqays 0.

Value: 5

VideoSizeChanged

Triggered by video renderers when the video size changed.

data: the new video width/height (the HiWord represents the new width; the LoWord represents the new height).

Value: 6
