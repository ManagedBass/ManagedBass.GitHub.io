Class BassDsd

BassDsd is a BASS addon enabling the playing of DSD (Direct Stream Digital) data in DSDIFF and DSF containers.

DefaultFrequency

The default sample rate when converting to PCM.

Declaration
public static int DefaultFrequency { get; set; }
Property Value
Int32

Remarks

This setting determines what sample rate is used by default when converting to PCM. The rate actually used may be different if the specified rate is not valid for a particular DSD rate, in which case it will be rounded up (or down if there are none higher) to the nearest valid rate; the valid rates are 1/8, 1/16, 1/32, etc. of the DSD rate down to a minimum of 44100 Hz.

The default setting is 88200 Hz. Changes only affect subsequently created streams, not any that already exist.

DefaultGain

The default gain applied when converting to PCM.

Declaration
public static int DefaultGain { get; set; }
Property Value
Int32

Remarks

This setting determines what gain is applied by default when converting to PCM. Changes only affect subsequently created streams, not any that already exist. An existing stream's gain can be changed via the DSDGain attribute.

The default setting is 6dB.
