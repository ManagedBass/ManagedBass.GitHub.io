ClientConnectProcedure Delegate

User defined client connection notification callback function.

Namespace: ManagedBass.Wma
Syntax
public delegate void ClientConnectProcedure(int Handle, bool Connect, string IP, IntPtr User);
Parameters
Handle
Int32

The encoder handle (as returned by EncodeSetNotify(Int32, ClientConnectProcedure, IntPtr)).
Connect
Boolean

The client's IP address... "xxx.xxx.xxx.xxx:port".
IP
String

The client is connecting?
User
IntPtr

The user instance data given when EncodeSetNotify(Int32, ClientConnectProcedure, IntPtr) was called.
Remarks

A client connection notification can be used to keep track of who's connected, where they're from, and for long they've been connected.
