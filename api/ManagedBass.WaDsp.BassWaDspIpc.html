Enum BassWaDspIpc

Communication to Winamp is done via the classic Win32 Message API. Most DSP plugins use this to ask for certain values. These definitions are the most commonly used messages, which might be handled by the fake winamp window which we create.

Namespace: ManagedBass.WaDsp
Assembly: ManagedBass.WaDsp.dll
Syntax
public enum BassWaDspIpc : int

Fields

GetInfo

Returns info about the current playing song. The value it returns depends on the value of 'mode' (wParam): 0 - SampleRate (i.e. 44100), 1 - Bitrate (i.e. 128), 2 - Channels (i.e. 2)

int inf=SendMessage(hwnd_winamp,WM_WA_IPC,mode,BassWaDspIpc.GetInfo);
GetListLength

Returns the length of the current playlist, in tracks.

int length = SendMessage(hwnd_winamp,WM_WA_IPC,0,BassWaDspIpc.GetListLength);
GetListPosition

Returns the playlist position [index].

int pos=SendMessage(hwnd_winamp,WM_WA_IPC,0,BassWaDspIpc.GetListPosition);
GetOutputTime

Returns the position in milliseconds of the current song (mode = 0), or the song length, in seconds (mode = 1). Returns -1 if not playing or error.

int res = SendMessage(hwnd_winamp,WM_WA_IPC,mode,BassWaDspIpc.GetOutputTime);
GetPlaylistFile

Gets the filename of the playlist entry [index]. Returns a pointer to it. Returns null on error.

char *name=SendMessage(hwnd_winamp,WM_WA_IPC,index,BassWaDspIpc.GetPlaylistFile);
GetPlaylistTitle

Gets the title of the playlist entry [index]. Returns a pointer to it. Returns null on error.

char *name=SendMessage(hwnd_winamp,WM_WA_IPC,index,BassWaDspIpc.GetPlaylistTitle);
GetVersion

Version will be 0x20yx for winamp 2.yx. Versions previous to Winamp 2.0 typically use 0x1zyx for 1.zx versions.

int version = SendMessage(hwnd_winamp,WM_WA_IPC,0,BassWaDspIpc.GetVersion);
Ipc

Is defined as Win32's WM_USER an. When a message of this value arrives, we know, that we might handle it by ourself.
IsPlaying

Returns the status of playback. If it returns 1, it is playing. if it returns 3, it is paused. if it returns 0, it is not playing.

int res = SendMessage(hwnd_winamp,WM_WA_IPC,0,BassWaDspIpc.IsPlaying);
StartPlay

Using IPC_STARTPLAY is like hitting 'Play' in Winamp, mostly.

SendMessage(hwnd_winamp,WM_WA_IPC,0,BassWaDspIpc.StartPlay);
