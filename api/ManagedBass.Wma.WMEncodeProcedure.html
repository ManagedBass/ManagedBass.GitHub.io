WMEncodeProcedure Delegate

Encoded data processing callback function.

Namespace: ManagedBass.Wma
Syntax
public delegate void WMEncodeProcedure(int Handle, WMEncodeType Type, IntPtr Buffer, int Length, IntPtr User);
Parameters
Handle
Int32

The encoder handle (as returned by EncodeOpen(Int32, Int32, WMAEncodeFlags, Int32, WMEncodeProcedure, IntPtr)).
Type
WMEncodeType

The type of data to process
Buffer
IntPtr

The pointer to the data to process. The buffer can contain the sample data or the header. The sample data is in standard Windows PCM format - 8-bit samples are unsigned, 16-bit samples are signed.
Length
Int32

The number of bytes in the buffer.
User
IntPtr

The user instance data given when EncodeOpen(Int32, Int32, WMAEncodeFlags, Int32, WMEncodeProcedure, IntPtr) was called.
Remarks

When encoding begins, an initial header is given. When encoding is completed, an updated header is given (with the duration info, etc.). When encoding to a file (whether that's on disk or not), the initial header should be replaced by the updated one.
