WMAEncodeFlags Enum

WMA encoding flags.

Namespace: ManagedBass.Wma

Byte

8 bit sample data.

Value: 1

Default

Default encoding, no tags.

Value: 0

Encode24Bit

Enable 24-bit encoding.

Value: 32768

Float

32-bit floating-point sample data.

Value: 256

PCM

Save uncompressed PCM data.

Value: 65536
Remarks

When the sample data is floating-point (Float flag is used), the Encode24Bit flag is considered; 24-bit data is written with it and 16-bit without. The "bitrate" parameter is ignored, except that it should be non-0.

Professional

WMA Professional encoding.

Value: 16384

Queue

Queue data to feed encoder asynchronously

Value: 262144

RatesCBR

GetRates: get available CBR quality settings.

Value: 0

RatesVBR

GetRates: get available VBR quality settings.

Value: 65536

Script

Enable the specification of tags mid-stream (after encoding has begun).

Value: 131072

Source

Use a BASS channel as source (provided in Frequency parameter). Channels parameter is ignored. If the BASSenc add-on is loaded, then the DSPPriority setting is used to determine where in the channel's DSP chain the encoding is performed, otherwise priority -1000 is used.

Value: 524288

Standard

Standard WMA encoding.

Value: 8192

Unicode

File is a Unicode (16-bit characters) filename.

Value: -2147483648
